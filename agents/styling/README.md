## About how to style this website

The main.css file is ready for your brand. To add custom styles:

- **Global styles:** Add to `app/assets/css/main.css`
- **Component styles:** Modify Tailwind classes in the section components
- **Color scheme:** Apply brand colors throughout components where necessary
- **Typography:** Update font families in `main.css` and `nuxt.config.js`

Visit `/sections` to see all available sections and their current styling, then customize as needed.
