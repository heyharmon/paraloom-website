# Paraloom Website

A Nuxt 4 application built without TypeScript, featuring:

- **Tailwind CSS** - Utility-first CSS framework via `@nuxtjs/tailwindcss`
- **Nuxt Content** - File-based CMS for Markdown, JSON, YAML, and CSV via `@nuxt/content`
- **Nuxt Fonts** - Optimized font loading and management via `@nuxt/fonts`

---

## Commands

```bash
# Install dependencies
npm install

# Start development server (http://localhost:3000)
npm run dev

# Build for production
npm run build

# Preview production build locally
npm run preview
```

Check out the [Nuxt documentation](https://nuxt.com/docs/getting-started/introduction) to learn more.
